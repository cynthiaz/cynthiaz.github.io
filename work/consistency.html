Overview

Collaborated with development team memebers in UX Consistency Feature Team to resolve UI inconsistencies and also to improve usability in existing products. I provided design guideline and UI specifications for Configuration Management related features (OSLC delegated dialog, menus, icons and interactions).

Example 1: OSLC Configurations Dialog (Open Services for Lifecycle Collaboration)

The OSLC dialog shows a custom delegated user interface which allows for searching and selecting configurations provided by any selected application. The challenge was the delegated UI provided by all applications had inconsistent interfaces and interactions. My role was to improve the usability and polish the interface for each delegated UI to ensure the overall consistency.

Example 2: Configuration Context Menu

Configuration context menu surfaces configuration related action which allow users to define their configuration context; create different versions for their artifacts and edit changes. Through usability testings, we discovered the menu exposed a lot of visual noise which made reading difficult, and many users couldn't find the right commands. Therefore the objectives were to improve its readability, and to provide necessary information to inform users about their workspace context.
