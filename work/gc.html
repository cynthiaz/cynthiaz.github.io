Overview

Global Configuration Manager is an application used to help systems and engineering organizations to effectively manage, organize and reuse components that contain artifacts across the development lifecycle. It is part of IBM's Configuration Management solution to support product line engineering.

  • Engaged in defining user flows and creating UX storyboards.
  • Created low to high fidelity UI mockups and icons to support playbacks and implementations
  • Ran usability tests with customers to identify and address usability issues
